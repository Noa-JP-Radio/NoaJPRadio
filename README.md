# Noa JP Radio - Landing Page

A minimalist, Notion-inspired landing page for the "Noa JP Radio" language learning podcast and newsletter.

## Features
- **Clean Design**: Minimalist black-and-white aesthetic.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **Bilingual**: Supports both English (`index.html`) and Japanese (`index_ja.html`).
- **Interactive**: Notion-style feature tabs and platform selection modal.
- **Embedded Audio**: Seamless integration with Substack and major podcast platforms.

## Project Structure
- `index.html`: English landing page (entry point).
- `index_ja.html`: Japanese landing page.
- `styles.css`: Core design system and responsive styles.
- `images/`: Brand assets and custom icons.

## Deployment
This project is designed to be hosted on **GitHub Pages**, **Netlify**, or **Vercel**.

### GitHub Pages Setup
1. Push this folder to a new GitHub repository.
2. Go to **Settings > Pages**.
3. Select the `main` branch and `/root` folder.
4. Your site will be live at `https://<username>.github.io/<repo-name>/`.
